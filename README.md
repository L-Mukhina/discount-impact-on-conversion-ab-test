# Discount Impact on Conversion — A/B Test Analysis

## Project Description
This project analyzes an A/B test evaluating the impact of a 50% discount on subscription conversion rates.

Group A: Subscription offer for $4.99  
Group B: Subscription offer for $4.99 with a 50% discount label

## Analysis Included
- User count in both groups
- Conversion count and conversion rate
- Test duration calculation
- Statistical hypothesis testing
- p-value calculation
- 95% confidence intervals
- Conversion dynamics over time visualization

## Tools Used
- Python
- Pandas
- Matplotlib
- Statistical hypothesis testing

## Files
- `ABTestAnalysis.ipynb` — full analysis with code
- `ABTestAnalysis.pdf` — final report version
